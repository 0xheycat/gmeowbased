## 🌟 What is GMEOWBASED?

GMEOWBASED is a **gamified quest platform** built on Farcaster that enables users to:

- **Complete Quests** - Earn XP and badges by completing challenges
- **Join Guilds** - Team up with others for group achievements
- **Mint NFTs** - Collect achievement badges and rank cards
- **Compete** - Climb leaderboards across multiple chains
- **Interact** - Engage with an intelligent bot for assistance

Built with **Next.js**, **Supabase**, **Base**, and **Farcaster frames**.

## 🎯 Key Features

### ✨ Gold Holographic XP Overlay
Stunning Yu-Gi-Oh! inspired XP notifications with particle effects and sound.

### 🤖 Intelligent Bot
Auto-replies with contextual frames based on user intent detection.

### 🎴 NFT Minting
Mint badges directly from Farcaster frames using EIP-155 format.

### 🏆 Multi-Chain Leaderboards
Compete globally or by chain (Base, Optimism, and more).

### 📊 Quest Wizard
Create custom quests with an intuitive multi-step interface.

## 🛠️ Tech Stack

- **Frontend**: Next.js 14, React 18, TailwindCSS
- **Backend**: Next.js API Routes, Supabase
- **Blockchain**: Wagmi, Viem, Base (EVM chains)
- **Farcaster**: Neynar API, Frames v1.2
- **Documentation**: Nextra, MDX
- **Testing**: Vitest, Playwright
- **Deployment**: Vercel

## 📦 Installation

```bash
# Clone the repository
git clone https://github.com/0xheycat/gmeowbased.git

# Install dependencies
pnpm install

# Set up environment variables
cp .env.example .env.local

# Run development server
pnpm dev
```

Visit [http://localhost:3000](http://localhost:3000) to see your app.
