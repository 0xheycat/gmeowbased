# Project Instructions — Gmeowbased Frame Development

## 🚨 CRITICAL: GI-15 Audit Required

Before ANY release touching frames, badges, scoring, or share/embed logic, you MUST:

1. **Run full GI-15 audit** (see `docs/maintenance/GI-15-Deep-Frame-Audit.md`)
2. **Generate audit report** (save to `docs/maintenance/reports/GI-15-AUDIT-REPORT-YYYYMMDD.md`)
3. **Get approval sign-off** from:
   - Product owner
   - Tech lead
   - Security owner
4. **Pass all 10 acceptance criteria** (no exceptions)
5. **Run Playwright test suite** (8 test groups, all must pass)
6. **Manual Warpcast testing** (mobile + desktop, see `STAGING-WARPCAST-TESTS.md`)

**Failure to complete GI-15 audit will block production deployment.**

## Quality Gates Hierarchy

### Required for ALL PRs:
- **GI-7:** Code Review & Testing
  - TypeScript compilation passes (`pnpm tsc --noEmit`)
  - Test suite passes (`pnpm test`)
  - ESLint passes (`pnpm lint`)
  - 2+ code reviewers approve

- **GI-8:** Security Controls
  - All inputs sanitized (use `lib/frame-validation.ts`)
  - Rate limiting enforced (60 req/min per IP)
  - HTTPS-only URLs (production)
  - No sensitive data in error messages

### Additional for Frame/Badge/Scoring PRs:
- **GI-9:** Frame Metadata Validation
- **GI-10:** Image Compliance (3:2 ratio frames, 1.91:1 OG)
- **GI-11:** URL Safety (no `/api/frame` exposure)
- **GI-12:** Performance (< 1s frame render, < 500ms OG)
- **GI-13:** Error Handling (graceful fallbacks)
- **GI-14:** Security Hardening (no secrets, CSP headers)
- **GI-15:** Deep Frame/MiniApp Coherence Audit ← **CRITICAL**

See `docs/maintenance/GI-7-to-GI-15-OVERVIEW.md` for full details.

## Validation Functions

**Always use these functions for input sanitization:**

```typescript
// lib/frame-validation.ts
import {
  sanitizeFID,        // Validate FID (1 to 2^31-1)
  sanitizeQuestId,    // Validate quest ID (0 to 999999)
  sanitizeChainKey,   // Validate chain (base, optimism, celo, unichain, ink)
  sanitizeUrl,        // Validate URL (HTTPS, max 1024 chars)
  sanitizeSplashImageUrl,  // Validate splash URL (max 32 chars)
  sanitizeButtons,    // Validate buttons (legacy Frames v1: max 4, titles ≤ 32 chars)
} from '@/lib/frame-validation'

// lib/miniapp-validation.ts
import {
  validateHexColor,       // Validate hex color (#RGB, #RRGGBB, #RRGGBBAA)
  validateMiniAppEmbed,   // Validate full MiniApp embed structure (1 button only)
  buildMiniAppEmbed,      // Safe embed construction (requires actionName)
} from '@/lib/miniapp-validation'
```

**⚠️ IMPORTANT - MCP-Verified Specifications (Nov 19, 2025):**
- **Mini App Embed:** Has ONE button only (singular), not 4
- **Action Types:** Only `launch_frame` and `view_token` are valid
- **Action Name:** REQUIRED field (Mini App name)
- **Invalid Types:** `link`, `post`, `mint` are NOT valid in Mini App Embed spec
- **Legacy Frames v1:** sanitizeButtons (max 4) is for backward compatibility only

**Example usage:**
```typescript
export async function GET(req: Request) {
  const { searchParams } = new URL(req.url)
  
  // ✅ Sanitize inputs
  const fid = sanitizeFID(searchParams.get('fid'))
  const questId = sanitizeQuestId(searchParams.get('questId'))
  const chain = sanitizeChainKey(searchParams.get('chain'))
  
  if (!fid || !questId || !chain) {
    return new Response('Invalid parameters', { status: 400 })
  }
  
  // ✅ Rate limiting
  const clientIp = getClientIp(req)
  const { success } = await rateLimit(clientIp, apiLimiter)
  if (!success) {
    return new Response('Rate limit exceeded', { 
      status: 429,
      headers: { 'retry-after': '60' }
    })
  }
  
  // ... rest of handler
}
```

## Image Requirements

### Frame Images (3:2 ratio)
- **Dimensions:** 1200×800 pixels
- **Format:** PNG or JPEG
- **Size:** < 1MB (< 500KB preferred)
- **URL:** HTTPS absolute, max 1024 chars

### OG Images (1.91:1 ratio)
- **Dimensions:** 1200×630 pixels
- **Format:** PNG or JPEG
- **Size:** < 1MB (< 500KB preferred)
- **URL:** HTTPS absolute, max 1024 chars

### Splash Images (1:1 ratio)
- **Dimensions:** 200×200 pixels
- **Format:** PNG RGB (no alpha channel)
- **Size:** < 100KB
- **URL:** Max 32 chars (relative or absolute)

### Icon Images (1:1 ratio)
- **Dimensions:** 1024×1024 pixels
- **Format:** PNG
- **Size:** < 200KB
- **URL:** HTTPS absolute

**Verification:**
```bash
identify public/frame-image.png
# Expected: public/frame-image.png PNG 1200x800 ...

identify public/splash.png
# Expected: public/splash.png PNG 200x200 ...
```

## Button Requirements

**⚠️ CRITICAL - MCP-Verified (November 19, 2025):**

### Mini App Embed (Modern - fc:miniapp:frame)
- **Button Count:** ONE button only (singular `button` object, not array)
- **Action Types:** ONLY `launch_frame` and `view_token` (MCP spec)
- **Action Name:** REQUIRED field (Mini App name)
- **Invalid Types:** `link`, `post`, `mint` are NOT in Mini App Embed spec
- **Title Length:** ≤ 32 characters
- **Target URL:** HTTPS absolute, max 1024 chars (optional)

### Legacy Frames v1 (Deprecated - fc:frame)
- **Max buttons:** 4 per frame (backward compatibility)
- **Action types:** `link`, `launch_frame`, `view_token`
- **Use sanitizeButtons:** Validates legacy frames only

**Example (Mini App Embed):**
```typescript
import { validateMiniAppEmbed, buildMiniAppEmbed } from '@/lib/miniapp-validation'

// Build with required action name
const embed = buildMiniAppEmbed({
  imageUrl: 'https://example.com/image.png',
  buttonTitle: 'Open App',
  actionType: 'launch_frame',
  actionName: 'My Mini App',  // REQUIRED
  actionUrl: 'https://miniapp.example.com'
})

// Validate
const { isValid, errors } = validateMiniAppEmbed(embed)
```

**Example (Legacy Frames v1):**
```typescript
import { sanitizeButtons } from '@/lib/miniapp-validation'

const { buttons, truncated, invalidTitles } = sanitizeButtons(rawButtons)
if (truncated) {
  console.warn(`Truncated ${rawButtons.length} buttons to 4`)
}
```

## Security Checklist

### Server Routes (`/app/api/*`)
- [ ] No client-side imports (`window`, `document`, `navigator`)
- [ ] No React client hooks (`useState`, `useEffect`, `useRouter`)
- [ ] All inputs sanitized
- [ ] Rate limiting enforced
- [ ] HTTPS-only URLs
- [ ] Error messages safe (no sensitive data)

### Client Components (`/components/*`)
- [ ] No server-only imports (database, secrets)
- [ ] No direct API calls (use tRPC/server actions)
- [ ] Environment variables prefixed `NEXT_PUBLIC_`

### Assets (`/public/*`)
- [ ] Fonts bundled locally (no CDN)
- [ ] All images < 1MB
- [ ] HTTPS URLs only
- [ ] Cache headers configured
