## Security Rules

### Input Validation
- Sanitize all user inputs (use Zod schemas)
- Parameterized queries only (prevent SQL injection)
- Validate FIDs, cast IDs, URLs before use
- Check origin for frame requests

### Rate Limiting
- API: 60 req/min per IP (Upstash Redis)
- Bot: 5 req/min per FID (in-memory)
- Neynar API: 500 req/5min (SDK handles)

### Authentication
- Admin: JWT-based session cookies
- Wallet: SIWE (Sign-In with Ethereum)
- Webhooks: HMAC SHA-512 signature verification
