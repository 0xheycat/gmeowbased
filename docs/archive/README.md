```
Active Work

Root Documentation (FOUNDATION-REBUILD-ROADMAP.md, CURRENT-TASK.md)
    ↓
Completion Reports (/docs/phase-reports/)

Historical Archive (/docs/archive/)
```

**Example**: Task 8.5 (Quest Creation)
1. **Planning**: `docs/planning/TASK-8.5-QUEST-CREATION-PLAN.md` (active)

4. **Summary**: Updated in `FOUNDATION-REBUILD-ROADMAP.md` (permanent)
5. **Archive**: Interim reports moved to `docs/archive/tasks/` (historical)
