## 📊 ELIGIBILITY PATTERNS

### Common Criteria Across Airdrops:
1. **Early Adoption**: Used protocol before snapshot date
2. **Transaction Volume**: Minimum number of transactions (usually 5-10)
3. **Time Period**: Active for 2-6 months
4. **Value Bridged**: Minimum $100-1000 in activity
5. **Unique Behaviors**: Smart contract interactions, governance participation

### Red Flags (Ineligible):
- ❌ Sybil behavior (multiple wallets from same source)
- ❌ Wash trading patterns
- ❌ Bot-like activity
- ❌ No organic usage after snapshot
