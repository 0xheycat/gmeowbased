### Merkle Distributor Contracts (Standard Pattern)
Most airdrops use this pattern:
```solidity
interface IMerkleDistributor {
    function claim(
        uint256 index,
        address account,
        uint256 amount,
        bytes32[] calldata merkleProof
    ) external;
    
    function isClaimed(uint256 index) external view returns (bool);
}
```

## 🔍 HOW TO VERIFY AIRDROP CONTRACTS

### 1. Official Sources Only
- ✅ Official protocol website
- ✅ Official Twitter/X announcement
- ✅ Verified block explorer (Etherscan, Basescan)
- ❌ NEVER trust random Telegram messages
- ❌ NEVER trust unofficial links

### 2. Check Contract Verification
```bash
# Example: Check on Etherscan
1. Go to etherscan.io or basescan.org
2. Search for contract address
3. Verify:
   - Contract is verified (green checkmark)
   - Contract source code is readable
   - Contract has official protocol name
   - Recent activity matches claims
```
