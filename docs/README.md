## 🛠️ Technical Stack

- **Frontend**: Next.js 14, React, TypeScript, Tailwind CSS
- **Smart Contracts**: Solidity (Base, Optimism, Celo, Unichain, Ink)
- **APIs**: Neynar (Farcaster), wagmi, viem
- **Database**: Supabase
- **Deployment**: Vercel, base.dev
