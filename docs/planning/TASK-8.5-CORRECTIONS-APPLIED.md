### 4. BASE POINTS vs XP Distinction ✅

**BEFORE**: Unclear points/XP usage
```typescript
// ❌ OLD (confusion between points and XP)
reward_xp: number // Which one? Points or XP?
reward_category: 'viral_xp' | 'base_points'
```

**AFTER**: Clear distinction between BASE POINTS and XP
```typescript
// ✅ NEW (clear distinction)
reward_points: number // BASE POINTS (from contract, escrowed from creator)
reward_xp: number // XP (backend logic, NOT escrowed)

// CRITICAL CLARIFICATIONS:
// BASE POINTS:
// - Source: leaderboard_calculations.base_points (from contract address)
// - SPENT by creator to create quests (escrowed in quest_creation_costs)
// - EARNED by users from quest completions
// - Used for leaderboard rankings

// XP:
// - Source: leaderboard_calculations.viral_xp (backend logic)
// - NOT spent by creator (generated by system)
// - Earned from social engagement (casts, likes, shares)
// - Used for rank/level progression ONLY
```

**Points Flow**:
```
Creator BASE POINTS → Escrow (quest_creation_costs) → Quest Rewards → User BASE POINTS
                ↓
          Quest Creation Cost
                ↓
          (baseCost + taskCost + rewardCost + badgeCost)
```
